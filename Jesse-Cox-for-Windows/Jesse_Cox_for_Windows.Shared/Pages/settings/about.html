About this app

This application was inspired by The Unofficial Jesse Cox Chrome Extension. It was not commissioned by Jesse Cox, and he is not responsible for bugs or feature requests.

This app was developed by Joshua Harms, a web application and Windows consultant specializing in ASP.NET, TypeScript, and Knockout. Joshua writes and talks about building real, practical windows and web applications that businesses actually need.

You can read about developing apps for Windows and the web at https://nozzlegear.com, or you can follow Joshua on Twitter at @nozzlegear.
